# Cubo 3D em Java 🎲

Projeto feito em Java com Swing que desenha e rotaciona um cubo 3D com projeção em perspectiva.

## ✨ Funcionalidades

- Projeção 3D para 2D com profundidade
- Rotação automática com Timer
- Faces coloridas
- Java puro, sem bibliotecas externas
